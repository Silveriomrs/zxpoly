ZX-Poly platform emulator

Version: 2.0.0
Author: Igor Maznitsa
Project page: https://github.com/raydac/zxpoly

ZX-Poly is an attempt to resolve ZX-Spectrum restriction and provide way to remove attribute clash issue. The idea of the platform was developed by Igor Maznitsa in april 1994, the first emulator was developed in 1998 and the current emulator was developed in 2014 to prove work of the idea.
The platform proves that it was possible for us to have a multi-CPU home platform in the middle of 80th and old games and utilities could be adapted either for full color mode with 16 colors per pixel or high resolution 512x384.

The emulator emulates listed components:

  1. ZX-Spectrum keyboard
  2. Beta-disk interface
  3. Kempston mouse (one click by mouse on the emulator form - trap mouse, ESC - free mouse)
  4. Kempston joystick
  5. Tape (only reading)

Keyboard shortcuts

  • F1 - this info
  • F3 - turbo mode
  • F4 - play/pause TAP file if loaded
  • F5 - show keyboard layout
  • F8 - make screenshot
  • F12- reset
