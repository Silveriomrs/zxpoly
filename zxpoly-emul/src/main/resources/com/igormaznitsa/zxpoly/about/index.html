ZX-Poly platform emulator

Version: 2.0.0
Author: Igor Maznitsa
Project page: https://github.com/raydac/zxpoly

ZX-Poly is attempt to jump over ZX-Spectrum limits and get way to have graphics without atribute clash. The idea of the platform was developed by Igor Maznitsa in april 1994. The first emulator was developed in 1998 and the current emulator was developed in 2014 to prove the idea.

By default the emulator emulates ZX-Poly platform but it is possible to turn on ZX-128 mode through item in Options menu.

Supported formats

  1. Snapshots: Z80, ZXP, SNA
  2. Disk format: TRD, SCL
  3. Tape format: TAP

Emulated components

  1. ZX-Spectrum keyboard (SHIFT = CAPS SHIFT, ALT = SYMBOL SHIFT)
  2. Beta-disk interface
  3. Kempston mouse (one click by mouse on the emulator form - trap mouse, ESC - free mouse)
  4. Kempston joystick (emulated by cursor keys of keypad!)
  5. Tape (only reading)

Keyboard shortcuts

  • F1 - this info
  • F3 - turbo mode
  • F4 - play/pause TAP file if loaded
  • F5 - show keyboard layout
  • F8 - make screenshot
  • F12- reset
  • CTRL+mouse wheel - screen scale

System properties

Emulator supports two system properties:
  • zxpoly.rom.path contains ROM file path
  • zxpoly.int.ticks number of INT ticks between screen refresh request
